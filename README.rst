QuickerUMLS
===========

QuickerUMLS is an extension of `QuickUMLS`_ tool providing faster, scalable,
and flexible concept extraction from medical narratives.


.. _`QuickUMLS`: https://github.com/Georgetown-IR-Lab/QuickUMLS
